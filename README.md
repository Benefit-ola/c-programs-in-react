# C Programs in React

A collection of programs originally written in C for my IT diploma — reimagined as interactive React web apps.

🔗 **Live site:** [c-programs-in-react.vercel.app](https://c-programs-in-react.vercel.app)

## Projects
- **Calculator** — arithmetic operations using a switch statement, input validation, step-by-step working
- **Quadratic Solver** — discriminant logic, two real roots / one repeated root / complex roots, live parabola graph

## Tech Stack
C · React · Tailwind CSS · Chart.js · Vite
